< !Doctype >
Conheça a tag < !Doctype >

A tag < !Doctype >, no tempo passado, quando o HTML era jovem (cerca de 1991/2), funcionava como links para um conjunto de regras que a página HTML tinha que seguir para ser considerada um bom HTML, o que poderia significar verificação automática de erros e outras coisas úteis. No entanto, atualmente, ninguém se importa com eles, e eles são apenas um artefato histórico que precisa ser incluído para que tudo funcione corretamente. < !Doctype > é a sequência mais curta de caracteres que conta como um doctype válido; atualmente, o < !Doctype > informa ao navegador a versão do HTML5. Por fazer parte da estrutura básica do HTML ele não aparecerá no navegador.

Código

<!Doctype>
<html>
<head>
<title> Projeto Código Livre</title>
</head>
<body>
<p> Primeiro Parágrafo. <p>
</body>
</html>

Tag < a >
Conheça a tag < a >

O elemento < a > é utilizado para criar links (âncoras) nas páginas web. O link faz conexão com páginas da mesma aplicação (links internos) ou em outros sites/aplicações (links externos). Para utilizar a âncora é necessário utilizar o atributo href para indicar o nome do arquivo e/ou link. Nessa tag é possível utilizar outro atributo importante, chamado target, esse atributo define em qual página o link será aberto.

Código para utilização < a >

Código

< a href="https://somas.org.br/" target=”_blank”> Somas < /a>

Resultado

Somas (esse link será aberto em outra página)
